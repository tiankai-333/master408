## 二、核心业务模块关系图

### 2.1 用户模块

```
t_user
  ├── id (PK)
  ├── user_name (用户名)
  ├── password (密码)
  ├── real_name (真实姓名)
  ├── user_level (年级/级别)
  ├── role (角色: 1-学生, 3-管理员)
  └── status (状态: 1-启用, 2-禁用)
```

**关联关系**:
- `t_user_token.user_id` → `t_user.id`
- `t_user_event_log.user_id` → `t_user.id`
- `t_exam_paper.create_user` → `t_user.id`
- `t_exam_paper_answer.create_user` → `t_user.id`
- `t_task_exam_customer_answer.create_user` → `t_user.id`

### 2.2 试卷模块

```
t_exam_paper (试卷)
  ├── id (PK)
  ├── name (试卷名称)
  ├── subject_id (学科)
  ├── paper_type (试卷类型: 1-固定, 4-时段, 6-任务)
  ├── grade_level (年级)
  ├── frame_text_content_id (试卷结构JSON)
  └── task_exam_id (关联任务试卷)
       ↓
       ↓ 包含
       ↓
t_exam_paper_question (试卷题目)
  ├── exam_paper_id (FK)
  ├── question_id (FK → t_question)
  └── item_order (题目顺序)
```

**关联关系**:
- `t_exam_paper.subject_id` → `t_subject.id`
- `t_exam_paper.frame_text_content_id` → `t_text_content.id`
- `t_exam_paper_question.exam_paper_id` → `t_exam_paper.id`
- `t_exam_paper_question.question_id` → `t_question.id`

### 2.3 答题模块

```
t_exam_paper_answer (试卷答题记录)
  ├── id (PK)
  ├── exam_paper_id (FK)
  ├── paper_type (试卷类型)
  ├── system_score (系统判分)
  ├── user_score (最终得分)
  └── status (状态: 1-待判分, 2-完成)
       ↓
       ↓ 包含
       ↓
t_exam_paper_question_customer_answer (题目答题记录)
  ├── exam_paper_answer_id (FK)
  ├── question_id (FK)
  ├── answer (学生答案)
  ├── text_content_id (简答/填空答案JSON)
  ├── customer_score (得分)
  └── do_right (是否正确)
```

**关联关系**:
- `t_exam_paper_answer.exam_paper_id` → `t_exam_paper.id`
- `t_exam_paper_answer.create_user` → `t_user.id`
- `t_exam_paper_question_customer_answer.exam_paper_answer_id` → `t_exam_paper_answer.id`
- `t_exam_paper_question_customer_answer.question_id` → `t_question.id`

### 2.4 题目模块

```
t_question (题目)
  ├── id (PK)
  ├── question_type (题型: 1-单选, 2-多选, 3-判断, 4-填空, 5-简答)
  ├── subject_id (学科)
  ├── grade_level (年级)
  ├── score (分数)
  ├── difficult (难度: 1-5)
  ├── correct (正确答案)
  └── info_text_content_id (题目详细信息JSON)
```

**关联关系**:
- `t_question.subject_id` → `t_subject.id`
- `t_question.info_text_content_id` → `t_text_content.id`
- `t_exam_paper_question.question_id` → `t_question.id`
- `t_exam_paper_question_customer_answer.question_id` → `t_question.id`

### 2.5 AI模块

```
t_ai_prompt_template (AI提示词模板)
  ├── id (PK)
  ├── style (风格: default, feynman, plato, first-principles)
  ├── system_prompt (系统提示词)
  ├── user_prompt_template (用户提示词模板)
  └── knowledge_base_ids (关联知识库)

t_ai_knowledge_base (AI知识库)
  ├── id (PK)
  ├── category (分类: 方法论, 领域, 示例)
  ├── domain (领域)
  ├── content (内容)
  └── enabled (是否启用)

t_ai_usage_log (AI使用日志)
  ├── id (PK)
  ├── template_id (模板ID)
  ├── style (风格)
  ├── question (题目)
  ├── response (AI响应)
  ├── tokens_used (消耗Token)
  └── duration_ms (耗时)
```

- `POST /api/student/exam/paper/pageList` → t_exam_paper

- `POST /api/student/examPaperAnswer/submit` →

- `POST /api/student/question/analyze-question` → t_question, t_text_content
