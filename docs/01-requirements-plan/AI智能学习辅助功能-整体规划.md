# AI智能学习辅助功能 - 整体规划

## 📌 核心目标

为考试系统添加 **智能学习辅助功能**，让AI帮助学生更好地理解和掌握知识点。

---

## 🎯 四种AI讲解风格

| 风格标识 | 名称 | 特点 | 适用场景 |
|----------|------|------|----------|
| **default** | 标准解析 | 清晰专业，讲清楚知识点 | 常规学习、常规理解 |
| **feynman** | 费曼风格 | 通俗易懂，像给小白讲解 | 难以理解的概念、需要通俗解释 |
| **plato** | 苏格拉底式 | 启发提问，引导思考 | 培养思维能力的深度学习 |
| **first-principles** | 第一性原理 | 从本质出发，逐步推导 | 理解底层原理、追根溯源 |

**字段说明**：

- `user_prompt_template` - 用户提示模板，支持变量替换
- `reference_materials` - 参考材料（JSON格式）
- `temperature` - 温度参数
- `max_tokens` - 最大token数
- `enabled` - 是否启用
- `is_default` - 是否默认风格

### 2. 提示词设计 ✅

#### 2.1 Default（标准解析）
```
系统提示：你是一个专业的计算机考研408辅导老师。你的任务是解析题目，帮助学生理解知识点。请用清晰、专业的语言进行讲解。

用户模板：请解析以下408考研题目：...
```

#### 2.2 Feynman（费曼风格）
```
系统提示：你是一个善于用费曼学习法讲解的老师。费曼学习法的核心是：用最简单，最通俗易懂的语言，把复杂的知识讲明白，就像给一个完全不懂的小白讲解一样。

用户模板：请用费曼学习法讲解以下408考研题目：...
```

#### 2.3 Plato（苏格拉底式）
```
系统提示：你是一个像苏格拉底一样的老师。你不直接告诉学生答案，而是通过提问和引导，让学生自己思考并发现问题。你相信"教育不是灌输，而是点燃火焰"。

用户模板：请用苏格拉底式提问法讲解以下408考研题目：...
```

#### 2.4 First-Principles（第一性原理）
```
系统提示：你是一个善于用第一性原理思考的老师。第一性原理的核心是：把问题分解到最基本的事实和不可再简化的真理，然后从这些基本点出发重新构建理解。

用户模板：请用第一性原理讲解以下408考研题目：...
```

## 🎯 核心使用场景

### 场景1：学生做题时获得AI解析
```
流程：
1. 学生做题并提交
2. 查看正确答案
3. 点击"AI解析"按钮
4. 选择讲解风格（默认/费曼/苏格拉底/第一性）
5. 等待AI生成解析
6. 查看个性化讲解内容
7. 对解析进行评分
```

### 场景2：复习时获取知识总结
```
流程：
1. 选择科目（数据结构/计算机网络/操作系统/组成原理）
2. 选择章节或知识点
3. 选择讲解风格
4. 获取该知识点的深度讲解
5. 建立完整的知识体系
```

### 场景3：调整AI参数优化效果
```
流程：
1. 查看解析结果
2. 对结果进行评分（1-5星）
3. 可选：填写反馈意见
4. 系统记录反馈
5. 管理员根据反馈调整提示词参数
```
