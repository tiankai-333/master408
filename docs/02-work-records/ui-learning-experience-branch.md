# UI Learning Experience Branch

## 分支

```text
codex/ui-learning-experience
```

## 设计目标

这个分支用于统一学生端的学习体验 UI，把页面从“功能入口堆叠”改成“学习工作台”。

核心方向：

- 首页动态化，突出 RAG、四种 Skill、Agent 学习档案和真题入口。
- 知识图谱页从力导向图改为知识点目录，避免大量节点堆叠导致不可读。
- 408 统考卷虽然是综合卷，但统计和学习建议要按数据结构、组成原理、操作系统、计算机网络分科表达。
- AI Skill 选择从线性列表改成四宫格，并支持设置默认视角。
- RAG 讲解、模拟出题、题目识别和关联真题应围绕选中的知识点组织。

## 已完成 UI 改动

### 1. 动态首页

文件：

```text
source/vue/xzs-student/src/views/dashboard/index.vue
```

改动：

- 使用动态 Hero 替代旧轮播图。

### 2. 知识工作台

文件：

```text
source/vue/xzs-student/src/views/knowledge-graph/index.vue
```

改动：

- 从 ECharts 力导向图改为“知识点目录 + 知识链接 + RAG 操作区”。
- 左侧展示分科正确率，而不是只展示综合正确率。
- 左侧底部展示四宫格 Skill，默认选中常规解析，并允许切换默认。
- 中间按学科分组展示知识点目录，支持搜索和折叠。
- 右侧展示知识点详情、下级知识链接、关联真题、RAG 讲解、模拟出题和题目识别入口。
